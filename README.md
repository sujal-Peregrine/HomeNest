# Landlord App API (Fastify + MongoDB)

Features:
- Email & Google (stub) auth
- Properties, Units (flats/rooms)
- Tenants & documents (metadata)
- Leases (assign tenant to unit)
- Monthly Rent Periods (billing) with penalties
- Payments
- Dashboard rollups

## Quickstart
```bash
npm install
cp .env.example .env
npm run seed
npm run dev
```
